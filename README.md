# gcnr
Gas Core Nuclear Reactor

Reactor power:
```math
\frac{dP}{dt}=\frac{\rho-\beta_{eff}}{l_0} P + \sum_{i=1}^{6} \lambda_i C_i
```
Precursors of delayed neutrons:
```math
\frac{dC_i}{dt}=\frac{\beta_i}{l_0} P - (\lambda_i + \lambda_r) C_i
```

Fuel energy:
```math
C_v M \frac{dT}{dt}=P + \gamma (T_0^4 - T^4)
```
